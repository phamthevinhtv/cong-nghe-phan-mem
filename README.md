## Đồ án kết thúc học phần Công nghệ phần mềm

### Đề tài: Website đăng ký khóa học

**Sinh viên thực hiện:**
- Phạm Thế Vinh  
- Nguyễn Phúc Vinh

**Công nghệ sử dụng:**
- Nodejs
- Express
- MySQL
- Reactjs

**Kiến trúc:**
- Kiến trúc client/server tách biệt Frontend và Backend.
- Frontend và Backend giao tiếp thông qua API.
